# Smart Requirement — Project Presentation

---

## คืออะไร?

**Smart Requirement** คือระบบรับและจัดการความต้องการจากลูกค้า (Lead Management System)
สำหรับบริษัทรับทำซอฟต์แวร์ / Freelancer / Software House

ลูกค้ากรอกฟอร์มออนไลน์ → ทีมงานได้รับแจ้งเตือนทันที → จัดการงานผ่าน Dashboard

---

## ปัญหาที่แก้ได้

### ก่อนมีระบบนี้
- ลูกค้าทัก LINE / Facebook แล้ว **หาย** ไม่มีการติดตาม
- ไม่รู้ว่าใครรับเรื่องไปแล้ว หรือยังไม่มีใครรับ
- ข้อมูลลูกค้ากระจายอยู่หลายที่ หาย้อนหลังไม่ได้
- ไม่รู้ว่างานไหน "ปิดได้" หรือ "หลุดไป" แล้ว

### หลังมีระบบนี้
- ลูกค้ากรอกฟอร์มครบ → ทีมได้รับ **Telegram แจ้งเตือนทันที**
- ทุก Lead อยู่ใน Dashboard ไม่มีหาย
- ติดตามสถานะได้ทุกขั้นตอน
- เห็นภาพรวมธุรกิจได้ชัดเจน

## ฟังก์ชันทั้งหมด

### หน้าฟอร์มลูกค้า

| ฟังก์ชัน | รายละเอียด |
|---|---|
| กรอกข้อมูลพื้นฐาน | ชื่อ, เบอร์โทร |
| เลือกประเภทงาน | 6 ประเภท บอกชัดเจนว่าแต่ละอันคืออะไร |
| งบประมาณอัจฉริยะ | งบเปลี่ยนตามประเภทงานที่เลือก ไม่งงแน่นอน |
| เลือก deadline | กำหนดระยะเวลาที่ต้องการ |
| ใส่เว็บอ้างอิง | แนบลิ้งเว็บที่ชอบ (ไม่บังคับ) |
| บอกรายละเอียด | อธิบายแบบชาวบ้าน ไม่ต้องใช้ภาษาเทคนิค |
| มีระบบเก่าหรือไม่ | เลือกได้ว่ามีระบบเดิมอยู่แล้วหรือเปล่า |
| Validation | ตรวจสอบข้อมูลแบบ real-time ก่อนส่ง |
| ป้องกัน spam | ส่งซ้ำไม่ได้ภายใน 3 นาที |

### Admin Dashboard

| ฟังก์ชัน | รายละเอียด |
|---|---|
| Login | เข้าระบบด้วย Email + Password |
| Analytics | ดูตัวเลข Lead แบบ real-time |
| กราฟ Analytics | กราฟเส้นแนวโน้ม Lead เลือกช่วง 7 / 30 / 90 วัน + แผนภูมิแท่งประเภทงาน + แผนภูมิแท่งงบประมาณ |
| ค้นหา | ค้นชื่อหรือเบอร์โทรได้ทันที |
| กรองข้อมูล | กรองตาม Status / ประเภทงาน / งบประมาณ |
| ดูรายละเอียด | คลิก Lead เพื่อดูข้อมูลครบทุกอย่าง |
| เปลี่ยน Status | กดปุ๊บเปลี่ยนเลย ไม่ต้องกดยืนยันซ้ำ |
| รีเฟรชข้อมูล | อัปเดตข้อมูลล่าสุดได้ตลอด |

### ระบบแจ้งเตือน Telegram

| ฟังก์ชัน | รายละเอียด |
|---|---|
| แจ้งเตือนทันที | มี Lead ใหม่ปุ๊บ Telegram เด้งทันที |
| ข้อมูลครบ | ชื่อ, เบอร์, ประเภทงาน, งบ, deadline, รายละเอียด |

## สถานะ Lead (Status Tracking)

```
ใหม่  →  ติดต่อแล้ว  →  กำลังดำเนินการ  →  ปิดงานสำเร็จ
                                         →  ไม่ได้งาน
```

| Status | ความหมาย |
|---|---|
| ใหม่ | Lead เพิ่งเข้ามา ยังไม่มีใครรับ |
| ติดต่อแล้ว | โทรหาหรือทักไปแล้ว |
| กำลังดำเนินการ | อยู่ระหว่างเสนอราคา / คุยรายละเอียด |
| ปิดงานสำเร็จ | ได้งานแล้ว |
| ไม่ได้งาน | ไม่ได้รับงานนี้ |

## วิธีใช้งาน — ทีละขั้นตอน

---

### ขั้นตอนที่ 1 — ลูกค้ากรอกฟอร์ม

**เข้าที่:** https://smart-requirement.vercel.app

**1.1** กรอก **ชื่อ-นามสกุล** หรือชื่อบริษัท

**1.2** กรอก **เบอร์โทรศัพท์** (9-10 หลัก)

**1.3** เลือก **ประเภทงาน** ที่ต้องการ
```
- เว็บไซต์แนะนำตัว / โชว์ผลงาน
- แอปมือถือ (iOS / Android)
- ระบบออนไลน์ / โปรแกรมใช้งานผ่านเน็ต
- ร้านค้าออนไลน์
- ระบบจัดการหลังบ้าน
- ไม่แน่ใจ / อื่นๆ
```

**1.4** เลือก **งบประมาณ** (จะเปลี่ยนให้เหมาะกับประเภทงานที่เลือก)

**1.5** เลือก **ระยะเวลาที่ต้องการ**

**1.6** กรอก **รายละเอียดที่ต้องการ** (เขียนแบบไหนก็ได้)

**1.7** กด **"ส่งข้อมูลขอใบเสนอราคา"**

→ จะเห็นหน้า "ขอบคุณมากครับ/ค่ะ!" พร้อม animation

---

### ขั้นตอนที่ 2 — ทีมงานรับแจ้งเตือน Telegram

ทันทีที่ลูกค้ากด Submit → **Telegram เด้งข้อมูลครบทันที**

```
🔔 Lead ใหม่เข้ามา!
👤 ชื่อ: สมชาย ใจดี
📞 โทร: 0812345678
🛠 ประเภทงาน: ร้านค้าออนไลน์
💰 งบประมาณ: 20,000 - 60,000 บาท
⏰ ต้องการภายใน: ภายใน 1-3 เดือน
📝 รายละเอียด: อยากได้เว็บขายเสื้อผ้า...
```

---

### ขั้นตอนที่ 3 — เข้า Admin Dashboard จัดการ Lead

**เข้าที่:** https://smart-requirement.vercel.app/admin

**3.1** Login ด้วย Email + Password ที่ตั้งไว้

**3.2** ดู **Analytics** ที่ด้านบน
```
ทั้งหมด | วันนี้ | ใหม่ | ติดต่อแล้ว | กำลังดำเนินการ | ปิดสำเร็จ | ไม่ได้งาน
```

**3.3** ค้นหา Lead ด้วยช่องค้นหา หรือกรองด้วย dropdown

**3.4** คลิก Lead ที่ต้องการ → เห็นข้อมูลครบ

**3.5** กดเปลี่ยน **Status** ตามสถานการณ์จริง → ปิด modal อัตโนมัติ

**3.6** กด **"รีเฟรช"** เพื่ออัปเดตข้อมูลล่าสุด
